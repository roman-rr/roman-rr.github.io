Offloads large tool outputs (>5k tokens) to S3 with presigned URLs. Uses tiktoken for accurate token counting.
threshold: 5000 tokens · format: head+tail preview
Tool Call Limit
Prevents infinite loops by limiting tool calls per conversation and per turn.
thread: 30 calls · run: 10 calls · behavior: continue
Tool Retry Middleware
Automatic retries for transient API failures with exponential backoff.
retries: 3 · backoff: 2.0x · initial: 1s
Todo List Middleware
Helps agents track progress on complex multi-step tasks with persistent state.
task tracking · progress monitoring · step management
Summarization Middleware
Automatically summarizes conversation history when context grows too large.
trigger: 50k tokens · keep: 20 messages · model: flash
Model Retry Middleware
Retries transient LLM API failures (429s, 503s, timeouts) before fallback.
retries: 3 · backoff: 2.0x · behavior: continue
Model Fallback Middleware
Multi-provider redundancy with automatic fallback to alternative LLMs.
primary → grok-4.1-fast → gpt-4o-mini
Usage Tracking Middleware
Fire-and-forget recording of tool calls, duration, and success. Manages trace and invocation lifecycle. Sub-agents run in tools_only mode, reading parent context via thread_id registry.
